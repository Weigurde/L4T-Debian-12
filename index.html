Welcome to L4T-Debian-12

L4T-Debian-12 is a fork of Debian Bookworm that is capable of running on Nintendo Switch. Since the OS options on Nintendo Switch are very limited, we have decided to build an up to date OS for use on your Switch.

Features

  • Up to date software
  • Easy installation
  • Support for Nintendo Switch
  • XFCE desktop environment included
  • LightDM (broken)
  • Working Nvidia drivers
  • Hardware accelerated Chromium
  • Xorg 1.20

Installation Instructions

To install L4T-Debian-12 on your Nintendo Switch, follow these steps:

  1. Download the installation files from github.
  2. Extract the files to a microSD card.
  3. Insert the microSD card into your Nintendo Switch.
  4. Follow the instructions on github to complete the installation.
